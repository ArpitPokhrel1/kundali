{% if locale=='ne' %}अस्त (Combustion){% else %}Combustion (Asta){% endif %}

{% if locale == 'ne' %} जुन ग्रह सूर्यबाट निश्चित दूरीभित्र छ, त्यो ‘अस्त’ हुन्छ। सूर्यको चमकले त्यस ग्रहको शक्ति निल्छ। अस्त ग्रहले फल कमजोर दिन्छ — विशेषतः बाह्य फल। आन्तरिक/आत्म-केन्द्रित फल भने मजबूत हुन सक्छ। {% else %} A planet within a certain distance from the Sun is "combust" (asta) — the Sun's brilliance overwhelms it. A combust planet's outer effects weaken; its inner / self-oriented effects can paradoxically strengthen. {% endif %}

{% if locale=='ne' %}वक्री (Retrograde){% else %}Retrograde (Vakri){% endif %}

{% if locale == 'ne' %} जब पृथ्वीको दृष्टिबाट कुनै ग्रह आकाशमा ‘उल्टो’ चलेजस्तो देखिन्छ, त्यसलाई वक्री भनिन्छ। केवल तारा-ग्रह (मङ्गल, बुध, गुरु, शुक्र, शनि) वक्री हुन्छन् — सूर्य र चन्द्र कहिल्यै हुँदैनन्। राहु र केतु सधैँ ‘वक्री जस्तै’ चल्छन्। {% else %} When a planet appears (from Earth) to move backwards through the zodiac, it is retrograde. Only the star-planets (Mars, Mercury, Jupiter, Venus, Saturn) ever go retrograde — Sun and Moon never do. Rahu and Ketu always move "as if" retrograde. {% endif %}

{% if locale == 'ne' %} परम्परागत व्याख्या: वक्री ग्रह तीव्र, गहन, र भित्री-केन्द्रित हुन्छ। शुभ ग्रह वक्री = फल आधा वा ढिलो; पीडक ग्रह वक्री = असर तीक्ष्ण। {% else %} Traditional take: a retrograde planet acts more intensely and inwardly. A benefic in retrograde may delay or muffle results; a malefic intensifies. {% endif %}

{% if locale=='ne' %}अयनांश (Ayanamsa){% else %}Ayanamsa{% endif %}

{% if locale == 'ne' %} आधुनिक खगोलशास्त्रको ‘सायन’ (वसन्त-सम्पात आधारित) र वैदिक ज्योतिषको ‘निरयन’ (तारा-स्थिर) राशिमा फरक छ — त्यो फरक नै अयनांश। हाम्रो साइटले लाहिरी अयनांश (नेपाल/भारतको सरकारी मानक) प्रयोग गर्छ — २०२६ मा करिब २४.२° को मान। {% else %} The difference between the modern tropical zodiac (anchored to the spring equinox) and the Vedic sidereal zodiac (anchored to fixed stars) is called the ayanamsa. This site uses the Lahiri ayanamsa (the official Indian/Nepali standard), which is around 24.2° in 2026. {% endif %}

{% if locale=='ne' %}विभाग कुण्डली (D9 — नवांश){% else %}Divisional charts (D9 — Navamsha){% endif %}

{% if locale == 'ne' %} जन्म कुण्डली (D1) मात्र होइन, हरेक राशिलाई ९ टुक्रामा बाँडेर नवांश (D9) बनाइन्छ — विवाह, सम्बन्ध, र ग्रहको ‘भित्री शक्ति’ हेर्न प्रयोग। अरू पनि — D7 (सन्तान), D10 (करियर), D12 (बुबा-आमा), D60 (सूक्ष्म फल) — हरेकले विशेष क्षेत्रको गहन व्याख्या दिन्छ। {% else %} Beyond the birth chart (D1), each rashi is divided into 9 to make the Navamsha (D9) — used for marriage, relationships, and a planet's "inner strength". Other divisions: D7 (children), D10 (career), D12 (parents), D60 (subtle results) — each gives finer reading of a specific area. {% endif %}

{% if locale == 'ne' %} हाल हाम्रो साइटले D1 (मूल कुण्डली) मात्र देखाउँछ। D9 भविष्यमा थपिने। {% else %} This site currently shows D1 (the birth chart) only. D9 navamsha is on the roadmap. {% endif %}
