{% if locale=='ne' %}२७ नक्षत्र (Nakshatras){% else %}The 27 Nakshatras{% endif %}

{% if locale == 'ne' %} नक्षत्र भनेको आकाशको २७ टुक्रा हुन् — हरेक १३°२०′ को (राशिभन्दा सूक्ष्म)। चन्द्र जुन नक्षत्रमा रहन्छ, त्यसैले व्यक्तिको जन्म नक्षत्र निर्धारण गर्छ। नक्षत्रले विंशोत्तरी महादशाको प्रारम्भ-स्वामी पनि तय गर्छ। {% else %} The 27 nakshatras (lunar mansions) divide the sky into 13°20′ slices — finer than the rashis. The nakshatra in which the Moon sits at birth becomes the Janma Nakshatra, which also fixes the starting lord of your Vimshottari Dasha. {% endif %}

{% if locale == 'ne' %} हरेक नक्षत्रको ४ पाद (३°२०′ प्रत्येक) हुन्छन्। नक्षत्रको स्वामी ग्रहले विंशोत्तरी क्रम तय गर्छ। {% else %} Each nakshatra has 4 padas (3°20′ each). The nakshatra's lord determines the Vimshottari order at birth. {% endif %}

{% if locale=='ne' %}विंशोत्तरी क्रम{% else %}Vimshottari order{% endif %}

{% if locale == 'ne' %} नक्षत्र-स्वामीहरू ९ ग्रहमा यसरी दोहोरिन्छन्: {% else %} The 9 nakshatra lords cycle in this order: {% endif %}

Ketu → Venus → Sun → Moon → Mars → Rahu → Jupiter → Saturn → Mercury

{% if locale == 'ne' %} अश्विनी = केतु, भरणी = शुक्र, कृत्तिका = सूर्य, ... हरेक ९ नक्षत्रमा यो ९-ग्रह क्रम तीन पटक दोहोरिन्छ (२७ नक्षत्र / ९ = ३ चक्र)। {% else %} Ashwini = Ketu, Bharani = Venus, Krittika = Sun, ... this 9-graha sequence repeats three times across the 27 nakshatras (27 / 9 = 3 cycles). {% endif %}
