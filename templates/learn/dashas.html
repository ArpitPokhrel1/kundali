{% if locale=='ne' %}विंशोत्तरी महादशा{% else %}Vimshottari Mahadasha{% endif %}

{% if locale == 'ne' %} विंशोत्तरी भनेको १२० वर्षको चक्र — ९ ग्रहले बाँडेर लिने अवधिहरू (महादशा)। हरेक महादशाले त्यस अवधिको जीवनलाई आफ्नो स्वामीको गुणले रङ्ग्याउँछ। जन्म समयमा चन्द्र कुन नक्षत्रमा थियो — त्यसले सुरुको दशा-स्वामी तय गर्छ। {% else %} Vimshottari is a 120-year cycle of nine planetary periods called Mahadashas. Each Mahadasha colors that stretch of life with its lord's qualities. The Moon's nakshatra at birth determines which lord's period you start with. {% endif %}

{% if locale=='ne' %}सुरु कसरी हुन्छ?{% else %}How does it begin?{% endif %}

{% if locale == 'ne' %} जन्म नक्षत्रको स्वामीले पहिलो महादशा दिन्छ। तर त्यो पूर्ण होइन — जन्म हुँदा त्यस नक्षत्रको कति बाँकी थियो, त्यतिकै महादशा बाँकी रहन्छ। उदाहरण: जन्म हुँदा चन्द्र अश्विनी नक्षत्रमा छ भने पहिलो महादशा केतुको हुन्छ; अश्विनीको आधा बाँकी थियो भने ७ × ½ = ३.५ वर्ष केतुको दशा रहन्छ। {% else %} The lord of your Janma Nakshatra gives the first Mahadasha — but only the UNUSED fraction of that nakshatra's period. Example: born with Moon in Ashwini → first Mahadasha is Ketu; if half of Ashwini was already traversed at birth, you get 7 × ½ = 3.5 years of Ketu. {% endif %}

{% if locale=='ne' %}उप-दशा (Antardasha){% else %}Sub-periods (Antardasha){% endif %}

{% if locale == 'ne' %} महादशालाई १२० वर्षकै अनुपातमा फेरि ९ ग्रहमा बाँड्दा अन्तरदशा निस्किन्छ। उदाहरण: १६ वर्षको गुरुको महादशामा गुरु-शुक्र अन्तरदशा २०/१२० × १६ ≈ २.६६ वर्ष। यसले सूक्ष्म समयको पूर्वानुमान दिन्छ। थप सूक्ष्म तह: प्रत्यन्तर (दशा / अन्तरदशा / प्रत्यन्तर)। {% else %} Each Mahadasha is further subdivided into 9 Antardashas (sub-periods) in the same 120-year proportion. Example: in a 16-year Jupiter Mahadasha, the Jupiter–Venus antardasha is 20/120 × 16 ≈ 2.66 years. This gives finer timing. Even finer: Pratyantar (Maha / Antar / Pratyantar). {% endif %}
