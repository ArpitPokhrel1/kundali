{% if locale=='ne' %}१२ राशि (Rashis){% else %}The 12 Rashis (Signs){% endif %}

{% if locale == 'ne' %} राशि भनेको आकाशको ३०° को टुक्रा हो। १२ राशिले मिलेर पूरा वृत्त (३६०°) बनाउँछन्। हरेक राशिको आफ्नो स्वामी ग्रह र विशेष स्वभाव हुन्छ। राशि गुणको तीन समूह (तत्त्व र भाव) मा पनि बाँडिन्छन्। {% else %} A rashi is a 30° slice of the sky. Twelve rashis make a full 360° circle. Each rashi has a ruling planet and a characteristic temperament. Rashis are also grouped by element (fire / earth / air / water) and modality (movable / fixed / dual). {% endif %}
