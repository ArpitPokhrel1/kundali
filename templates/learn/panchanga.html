{% if locale=='ne' %}पञ्चाङ्ग — पाँच अङ्ग{% else %}Panchanga — The Five Limbs{% endif %}

{% if locale == 'ne' %} पञ्चाङ्ग भनेको ‘पाँच अङ्ग’। एउटा क्षणको ‘गुण’ कस्तो छ — त्यो पाँच कुराबाट बुझिन्छ: तिथि, नक्षत्र, योग, करण, र वार। मुहूर्त (शुभ बेला छान्ने), नामकरण, विवाह, र पूजामा यी पाँचै कुरा हेरिन्छन्। {% else %} Panchanga means "five limbs". The Vedic way of describing the quality of any moment uses five components: Tithi, Nakshatra, Yoga, Karana, and Vara. They are used in muhurta (auspicious timing), naming ceremonies, marriages, and rituals. {% endif %}

1. {% if locale=='ne' %}तिथि (Lunar day){% else %}Tithi (Lunar day){% endif %}

{% if locale == 'ne' %}

तिथि भनेको चन्द्र र सूर्यबीचको कोणीय दूरीबाट निस्किने ‘चान्द्र दिन’। हरेक तिथि १२° बराबर हुन्छ। एक चान्द्र महिनामा ३० तिथि।

  • शुक्ल पक्ष (तिथि १–१५): चन्द्रमा बढ्दै जाने अवधि — सकारात्मक कामका लागि शुभ।
  • कृष्ण पक्ष (तिथि १६–३०): चन्द्रमा घट्दै जाने — आन्तरिक काम, साधना।

तिथिका नाम: प्रतिपदा, द्वितीया, तृतीया, चतुर्थी, पञ्चमी, षष्ठी, सप्तमी, अष्टमी, नवमी, दशमी, एकादशी, द्वादशी, त्रयोदशी, चतुर्दशी, पूर्णिमा/अमावस्या।

{% else %}

The Tithi is the "lunar day", computed from the angular distance between the Moon and Sun. Each tithi covers 12° of separation. There are 30 tithis in a lunar month.

  • Shukla Paksha (tithi 1–15): waxing Moon — auspicious for outward action.
  • Krishna Paksha (tithi 16–30): waning Moon — inner work, sadhana.

Tithi names: Pratipada, Dwitiya, Tritiya, Chaturthi, Panchami, Shashthi, Saptami, Ashtami, Navami, Dashami, Ekadashi, Dwadashi, Trayodashi, Chaturdashi, Purnima/Amavasya.

{% endif %}

{% if locale == 'ne' %} जुन नक्षत्र मा त्यस क्षणमा चन्द्र छ। २७ नक्षत्र छन्, हरेक १३°२०′ चौडाइको। हरेकको आफ्नो देवता र विषय हुन्छ। → सबै २७ नक्षत्रको तालिका {% else %} The nakshatra in which the Moon currently sits. There are 27 nakshatras, each 13°20′ wide. Each has its own deity and theme. → Full table of 27 nakshatras {% endif %}

3. {% if locale=='ne' %}योग (Sky-quality){% else %}Yoga (Sky-quality){% endif %}

{% if locale == 'ne' %}

योग = (सूर्यको देशान्तर + चन्द्रको देशान्तर), २७ नामहरूमा बाँडिएको। यसले एउटा क्षणको ‘समग्र शुभ/अशुभ’ भाव बताउँछ। ‘विष्टि’, ‘व्यतीपात’, ‘वैधृति’ जस्ता योगहरू सामान्यतया अशुभ; ‘सिद्धि’, ‘शुभ’, ‘ब्रह्म’ जस्ता शुभ।

{% else %}

Yoga is calculated as (Sun longitude + Moon longitude), divided into 27 named yogas. It describes the overall favorability of the moment. Some (like Vishti, Vyatipata, Vaidhriti) are considered inauspicious; others (Siddhi, Shubha, Brahma) auspicious.

{% endif %}

4. {% if locale=='ne' %}करण (Half-tithi){% else %}Karana (Half-tithi){% endif %}

{% if locale == 'ne' %}

करण = आधा तिथि (६° का दूरीहरू)। महिनामा ६० करण। यिनको ११ नाम छन् — सात (बव, बालव, कौलव, तैतिल, गर, वणिज, विष्टि) बारम्बार दोहोरिने, र चार (शकुनि, चतुष्पाद, नाग, किंस्तुघ्न) तय। मुहूर्तमा प्रयोग।

{% else %}

A karana is half a tithi (6° each). There are 60 karanas per lunar month, mapped to 11 names — seven recurring (Bava, Balava, Kaulava, Taitila, Garaja, Vanija, Vishti) and four fixed (Shakuni, Chatushpada, Naga, Kimstughna). Used in fine-grained muhurta timing.

{% endif %}
