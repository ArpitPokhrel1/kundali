{% if locale=='ne' %}उद्देश्य{% else %}Purpose{% endif %}

{% if locale == 'ne' %} {{ SITE_DOMAIN }} एक निःशुल्क, खुला-स्रोत वैदिक ज्योतिष उपकरण हो — नेपाली र अङ्ग्रेजी दुवै भाषामा। उद्देश्य: सटीक गणनासँगै शिक्षा-केन्द्रित व्याख्या दिनु, ताकि कुनै पनि जिज्ञासुले आफ्नो कुण्डली बुझ्न र क्रमशः ज्योतिष सिक्न पाओस्। {% else %} {{ SITE_DOMAIN }} is a free, open-source Vedic astrology tool — bilingual (Nepali + English). The aim is to provide accurate computation alongside education-first explanations, so any curious person can both understand their kundali and gradually learn Jyotish. {% endif %}

{% if locale=='ne' %}गणनाको आधार{% else %}Computation method{% endif %}

  • {% if locale=='ne' %}दृक् सिद्धान्त{% else %}Drik Siddhanta{% endif %}: {% if locale=='ne' %} Swiss Ephemeris (DE431 आधारित) ले उपआर्कसेकेन्ड शुद्धताको आधुनिक खगोलशास्त्रीय गणना। निरयन रूपान्तरण लाहिरी अयनांश (नेपाल/भारत सरकारी मानक) मार्फत। {% else %} Sub-arcsecond modern astronomical positions via Swiss Ephemeris (DE431-backed). Sidereal conversion using the Lahiri ayanamsa (the Indian/Nepali government standard). {% endif %}
  • {% if locale=='ne' %}सूर्य सिद्धान्त{% else %}Surya Siddhanta{% endif %}: {% if locale=='ne' %} शास्त्रको मूल पाठबाट सिधा कार्यान्वयन — महायुगमा प्रति-ग्रह क्रान्तिहरू, अहर्गण, मन्द र शीघ्र समीकरणहरू। {% else %} Implemented straight from the original text — revolutions per Mahayuga, ahargana, and the manda + sighra epicycle equations. {% endif %}
  • {% if locale=='ne' %}विक्रम संवत्{% else %}Bikram Sambat{% endif %}: {% if locale=='ne' %} प्रामाणिक नेपाली पञ्चाङ्ग डेटा (वि.सं. २०००–२०९९ कभर हुने)। {% else %} Verified Nepali panchanga calendar data (covers BS 2000–2099 ≈ AD 1943–2042). {% endif %}
  • {% if locale=='ne' %}स्थान खोज{% else %}Place search{% endif %}: OpenStreetMap Nominatim. {% if locale=='ne' %}समय क्षेत्र{% else %}Timezone{% endif %}: timezonefinder + IANA zoneinfo (DST-सहित)।

{% if locale=='ne' %}सीमाहरू{% else %}Limitations{% endif %}

  • {% if locale=='ne' %} जन्म समय गलत भएमा लग्न ४ मिनेटको त्रुटिले १° सर्न सक्छ — सकेसम्म सटीक समय राख्नुहोस्। {% else %} A 4-minute error in birth time can shift the Lagna by 1° — be as precise as possible. {% endif %}
  • {% if locale=='ne' %} वि.सं. १९४३ अघिको र २०४२ इस्वी पछिको मितिमा BS रूपान्तरण उपलब्ध छैन। {% else %} BS conversion unavailable before 1943 AD or after 2042 AD. {% endif %}
  • {% if locale=='ne' %} अहिले D1 (मूल कुण्डली) मात्र — D9 (नवांश), अन्तरदशा र योग पहिचान भविष्यमा थपिने। {% else %} Currently D1 (birth chart) only — D9 navamsha, antardasha sub-periods, and yoga detection are on the roadmap. {% endif %}
  • {% if locale=='ne' %} ज्योतिष व्याख्या शास्त्रीय प्रवृत्ति मात्र हो — भविष्यवाणी होइन। {% else %} Astrological readings describe classical tendencies — they are not predictions. {% endif %}
